## Perhitungan

### 1. Menentukan N
N diambil dari dua digit terakhir NIM:  
N = 03 → 3  

---

### 2. Menentukan a
a = floor(N / 10)  
a = floor(3 / 10) = 0  

---

### 3. Menentukan b
b = N mod 10  
b = 3 mod 10 = 3  

---

### 4. Menentukan K
K = a + b + 1  
K = 0 + 3 + 1 = 4  

---

### 5. Menentukan θ₀ (derajat)
θ₀ = (a × 30) + (b × 5)  
θ₀ = (0 × 30) + (3 × 5) = 15°  

---

### 6. Menentukan α (learning rate)
α = 0.001 × (b + 1)  
α = 0.001 × 4 = 0.004  

---

## Keterangan

Parameter ini akan digunakan pada seluruh perhitungan Soal 1 sampai Soal 7 sebagai bentuk personalisasi data sesuai NIM.
